# Netball Rulebook Oracle

## When to Use This Skill
- Validating player positions and zone constraints.
- Implementing scoring logic.
- Checking game play rules.

## Domain Specifics
- **Standard:** 7 players, 1pt per goal.
- **Suncorp Super Netball:** 7 players, 1pt per goal, 2pt goals in last 5 mins of each quarter in supershot zone.
- **FAST5:** 5 players, multi-point zones, Power Plays.
- **WINA:** Indoor specific variations for 6s and 7s.

## Instructions
Refer to the official documents in the `resources/` folder before generating any code that involves match rules or scoring validation.
